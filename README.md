# 🧠 *SMART STUDY ADVISOR*

## 📌 DESCRIPTION 
Smart Study Advisor is a Python-based project that helps students analyze their academic performance and provides personalized study recommendations. It uses rule-based logic (basic AI concept) to evaluate marks, suggest study hours, and give subject-wise advice.

---

## 🎯 FEATURES
-  Subject-wise performance analysis (Weak, Average, Good, Excellent)
-  Personalized study time suggestions
-  Intelligent subject-wise advice
-  Identification of weakest and strongest subjects
-  Graphical visualization using bar graph
-  Color-coded performance representation
-  Modular code structure (separate files for logic, main, and graph)

---

## 🛠 TECHNOLOGIES USED
- Python
- Matplotlib (for graph visualization)

---

## 📂 PROJECT STRUCTURE
Smart-Study-Advisor/

│── main.py # Main program (user interaction)

│── advisor.py # Logic for analysis and advice

│── graph.py # Graph visualization

│── data.csv # (Optional) Data storage

│── README.md # Project documentation

## ▶️ HOW TO RUN THE PROJECT

1. Install Python (if not already installed)

2. Install required library:

   pip install matplotlib(if not installed)

3. Run the program:

   python main.py
